FOV Mod for Breath of the Wild (Wii U)

Versions:

Wider FOV - Increases field of view (fovy: 0.8)

Narrower FOV - Decreases field of view for zoomed in look (fovy: -0.5)

Extreme Wide FOV - Fisheye effect (fovy: 2.0)

Installation:

(Tiramisu) For legacy SDCafiine: Copy content folder to sd:/wiiu/sdcafiine/00050000101C9400/

(Aroma) For new SDCafiine: Copy content folder to sd:/wiiu/sdcafiine/00050000101C9400/(folder with whatever name you want)/

This mod modifies GameRomCamera.sbactorpack. Only use one version at a time
